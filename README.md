
Interactive web application designed to help users understand and navigate traffic offenses and penalties. It provides a comprehensive and easy-to-use interface for browsing driving violations, their associated fines, and the points system (سلم النقط).

- **Interactive Offenses Table (جدول المخالفات):** Browse and search through detailed records of traffic offenses.
- **Points Matrix (سلم النقط & المنهجية):** Understand how different violations affect your driving license points.
- **Search & Filtering:** Quickly find specific penalties or browse by category using the integrated controls.
- **Responsive Navigation:** Designed with a modern, high-contrast visual theme localized in Arabic.

This project is intended for educational and utility purposes.
